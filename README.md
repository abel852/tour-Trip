# Ethio Hiking

A premium, responsive adventure travel platform focused on Ethiopian hiking, trekking, and camping experiences. Built with React and modern CSS, featuring immersive UI sections, destination showcases, and interactive adventure layouts inspired by Ethiopia’s natural landscapes.

---

## Features

- Modern responsive design
- Adventure destination showcase
- Hiking & trekking sections
- Camping experience highlights
- Interactive UI components
- Mobile-friendly layout
- Clean and modern interface

---

## Featured Destinations

- Wenchi Crater Lake
- Simien Mountains
- Bale Mountains
- Entoto Hills
- Erta Ale Adventure Tours

---

## 🛠️ Tech Stack

- React.js
- CSS3
- React Icons

---

## 🚀 Installation

### 1️. Clone the repository

```bash
git clone https://github.com/yourusername/ethiotrek.git
cd ethiotrek
```

### 2️. Install dependencies

```bash
npm install
```

### 3️. Install icons

```bash
npm install react-icons
```

### 4️.  Start the project

```bash
npm start
```
